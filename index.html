  • Valider les designs avec le équipe et les stakeholders

FrontEnd (Angular)

  • Installer Angular dans un nouveau projet
  • Créer une page d'accueil pour le IHM
  • Intégrer les éléments de l'interface utilisateur conçus précédemment
  • Tester la navigation et la fonctionnalité générale de la page

BackEnd (Python)

  • Installer Flask ou Django pour créer un serveur web
  • Créer une API pour stocker et récupérer les données du IHM
  • Intégrer l'API avec le FrontEnd Angular
  • Tester la communication entre les deux parties

Intégration et Tests

  • Intégrer les différentes parties du projet (IHM, FrontEnd, BackEnd)
  • Lancer des tests pour s'assurer que tout fonctionne correctement
  • Valider l'IHM avec le équipe et les stakeholders
